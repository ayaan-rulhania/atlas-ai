# Changelog

All notable changes to this project will be documented in this file.

## [2.5.0] - 2026-01-29
### Changed
- Documentation refreshed for **Thor 1.2** (see `docs/thor-1.2-brain-architecture.md`)
- Documentation refreshed for **R-series** weight tooling (see `scripts/compact_r_weights.py`)
- Version alignment across app + SDK metadata to reflect this release

### Fixed
- Reduced repo bloat by avoiding committed build artifacts (e.g. `node_modules/`, Python build outputs)

## [2.1.0] - 2026-01-03
### Changed
- Updated dependencies to latest stable versions across all packages
- Improved code quality and maintainability
- Enhanced documentation and version tracking
- Updated Electron to latest stable version (^27.0.0)
- Updated Flask and related dependencies to latest versions
- Improved error handling and stability

### Fixed
- Resolved dependency version conflicts
- Fixed minor bugs in path management
- Improved import resolution and module loading

### Security
- Updated dependencies to address security vulnerabilities
- Improved dependency management and version pinning

## [2.0.0] - 2026-01-02
### Added
- Major UI/UX overhaul with modern vertical navigation rail
- Orange theme and full-page backgrounds
- Responsive mobile design
- Enhanced tooltips and persistent welcome screen
- Embedded app modals
- Model improvements and optimizations

## [1.0.0] - 2026-01-01
### Added
- Initial public release under Apache-2.0.
- Thor 1.1 & Thor 1.0 model weights via Git LFS.
- React/Flask chat UI, Python and JS SDKs.
- Continuous learning brain, TrainX DSL.
- CI workflow, examples, documentation, marketing assets.
