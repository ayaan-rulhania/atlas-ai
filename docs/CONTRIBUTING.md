# Contributing to Atlas-AI

Thanks for taking the time to contribute!

## Getting Started
1. Fork the repository and create your branch from `main`.
2. Install dev deps with `pip install -r requirements.txt` and `npm install`.
3. Run tests with `pytest` and `npm test`.

## Pull Request Process
- Open a draft PR early for feedback.
- Ensure CI passes and add tests for new features.
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md).

## Good First Issues
Look for issues labeled **good first issue** or open a discussion if unsure.
