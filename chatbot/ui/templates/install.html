Install Atlas

Get the full Atlas experience as a native desktop application. Run Atlas offline, access it from your dock/taskbar, and enjoy a seamless desktop experience.

Download for macOS
Installation Note
After downloading, if macOS shows a security warning:
  1. When you see the security warning, click "Done"
  2. Go to System Settings > Privacy & Security
  3. Scroll down to the security section
  4. You should see a message about Atlas being blocked
  5. Click "Open Anyway" next to the Atlas message
This is normal for unsigned applications and only needs to be done once.
After downloading, run the installer and follow the setup wizard. Windows may show a security warning - click "More info" and then "Run anyway" if needed. This is normal for unsigned applications.
After downloading, make the AppImage executable: chmod +x Atlas-*.AppImage then run it. For .deb packages, install with: sudo dpkg -i Atlas-*.deb
