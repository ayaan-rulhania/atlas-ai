How to use Atlas

Getting Started

Atlas is an AI assistant powered by Thor models. Simply type your question or request in the input field and press Enter or click Send. You can ask questions, request help with tasks, generate content, and much more.

Quick Commands

/office or Load Office Suite
Opens the integrated office suite interface with productivity tools
/arcade or Load Game Suite
Opens the integrated arcade interface for interactive games
/image <description>
Generates and displays an image based on your description. You can refine with phrases like "Another angle", "Watercolor style", or "Make it red"

Features & Tips

  • ✓
    Think Deeper Mode — Toggle this for more comprehensive, thorough responses to complex queries. Great for detailed analysis and deep reasoning.
  • ✓
    Custom Gems — Create personalized AI models tailored to your needs. Access via the sidebar or Customize button.
  • ✓
    Voice Assistant (Poseidon) — Click the microphone icon to start voice conversations. Perfect for hands-free interactions.
  • ✓
    Model Selection — Switch between Thor 1.1 (latest) and Thor 1.0 (stable) from the model dropdown.
  • ✓
    Structured Output — Request bullets, steps, lists, or tables when you need organized information.
  • ✓
    Chat History — All your conversations are saved. Access them via the History button or sidebar.

Best Practices

  • • Be specific in your requests — the more details you provide, the better the response
  • • Use natural language — you don't need to use special syntax or commands
  • • Ask follow-up questions to refine responses or dive deeper into topics
  • • Take advantage of projects to organize related conversations with shared context
  • • Experiment with different models and tones to find what works best for your needs
